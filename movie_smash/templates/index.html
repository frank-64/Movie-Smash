Child Minding Website

HTML & CSS
JavaScript
PHP

This project was for my A-Level coursework in 2019. My client was a childminder who required a website to replace her paper-based system. I produced this website using HTML, CSS, JS and PHP. The website has the following functionality:

  • Login page
  • Registration page
  • User dashboard
    • View bookings
    • Pay for booking using integrated PayPal API
    • Make booking
    • View children
    • Add a child
    • Edit Info
  • Admin Account
    • View/edit/delete user bookings
    • View/edit/delete users
    • View/edit/delete all children
    • Timestamp child arrival for referral
Please check out the live website and it's functionality.
